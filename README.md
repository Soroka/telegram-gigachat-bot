# Telegram Bot для изменения стилистики постов через GigaChat API для AlfaBank

Бот для изменения стилистики постов с использованием GigaChat от Сбера. Сделан как техническое задание для AlfaBank.

## Что он делает и почему

- Принимает на вход название канала (и опционально ключевые слова через #)
- Принимает на вход ссылку на новостную статью
- Качает из канала несколько подходящих постов (можно только с ключевым словом, матчинг по подстроке глупый, но для хештегов его точно хватит)
- Перефразирует новостную статью в стиле канала, используя эти посты как пример в промпте
- Решила не фиксировать канал. Сделать, чтобы и канал, и тему можно было выбирать.
- Так интереснее. Но при плохом выборе темы и канала возможны проблемы.
- Бот хостится на Repl.it, чтобы repl.it его не гасил, туда ходит UptimeRobot и пингует flask app, который держит его живым

## Какие с ним возможны проблемы и почему

- Могут подобраться посты, которые Гигачат откажется перефразировать как пост на чувствительные темы - из-за цензуры
- Может не распарситься новостная статья. Я брала newspaper, какие-то форматы может не взять. Выбрала как меньшее зло - чтобы не брать из новостей левые тексты

## Использование ##

1. Начните диалог с ботом: `/start`
2. Отправьте команду `/post`
3. Введите канал для копирования стилистики, например @AlfaBank
4. Введите ссылку на новостную статью, которую нужно перефразировать
5. Получите сгенерированный пост!
